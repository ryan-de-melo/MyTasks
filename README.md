# MyTasks

**MyTasks** é uma aplicação web full stack para **gerenciamento de tarefas**,
 desenvolvida com foco em produção de API robusta, autenticação segura e 
 comunicação entre o frontend e a API.

O projeto utiliza **Spring** no backend, **React** no frontend, **JWT** 
para autenticação e **Docker** para virtualização do ambiente.

---

## Funcionalidades

- Cadastro e autenticação de usuários  
- Login com **JWT (JSON Web Token)**  
- Criação, listagem, atualização e remoção de tarefas  
- Tarefas associadas ao usuário autenticado
- Proteção de rotas no backend
- Comunicação frontend / backend via **Axios**
- Ambiente totalmente containerizado com **Docker**
